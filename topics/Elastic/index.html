Introduction ¶

The DFPT theory to compute the elastic, piezoelectric and internal strain tensors is presented in Hamann2005.

See the DFPT as well as the tutorial lesson_elastic. For the preliminary runs of ABINIT, see rfstrs, while for ANADDB, see elaflag for the elastic tensor, and piezoflag for the piezoelectric tensor.

Related Input Variables ¶

basic:

  • elaflag ELAstic tensor FLAG
  • instrflag INternal STRain FLAG
  • piezoflag PIEZOelectric tensor FLAG
